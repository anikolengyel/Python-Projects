Introduction

This project is a Flask aplication to download pictures of a webpage.

Setup

Clone this repository.
The directory contains:
- picture_downloader.py - the program
- static - contains the HTML and CSS files to the webpage
- this README.txt file

If you do not have the necessary programs, you will need to install the followings:
- Python3
- Python packages: flask

Running

- run the picture_downloader.py file
- open the start_page.html in you browser
